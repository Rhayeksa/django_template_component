# Django Template Component

Project yang akan digunakan sebagai referensi ketika pembuatan project menggunakan Django framework

## Teknologi

- Python 3.11.7
- Python 3.11.7-venv
- Django 5.0.1
- db.sqlite3

## Membuat Virtual Env

- Buat folder dan masuk direktori foldernya
- Buat atau copy file requirements.txt
- Run command :

  ```
  python3.11 -m venv .venv
  pip install --upgrade pip
  pip install -r ./requirements.txt
  ```
