# Django Template Component

Project yang akan digunakan sebagai referensi ketika pembuatan project menggunakan Django framework

## Teknologi

- Python 3.11.7
- Python 3.11.7-venv
- Django 5.0.1
- db.sqlite3

## Membuat Project dan Virtual Env

- Buat folder dan masuk direktori foldernya

```
mkdir my_first_django_project
cd ./my_first_django_project
```

- Buat atau copy file requirements.txt
- Run command :

  ```
  python3.11 -m venv .venv
  . ./.venv/bin/active
  pip install --upgrade pip
  pip install -r ./requirements.txt
  ```
